# LibraryProject

This is a basic Django project created for learning purposes under *Alx_DjangoLearnLab*.

## Steps Completed
1. Installed Django
2. Created a project named LibraryProject
3. Ran the development server
4. Explored the project structure

## Project Structure
- manage.py → Command-line utility to interact with the project.
- settings.py → Stores configuration for the project.
- urls.py → URL declarations (like a table of contents).
- wsgi.py & asgi.py → Deployment entry points.
